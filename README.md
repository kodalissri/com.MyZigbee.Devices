# Naresh Personal - Homey App

Add Support for Missing Devices

## Description

This Homey app provides support for various Zigbee devices that are not natively supported by Homey. It includes drivers for door/window sensors, temperature & humidity sensors, motion sensors, and smart knobs.

## Supported Devices

### Door/Window Sensor
- **SNZB-04** - eWeLink/Zbeacon Door/window Sensor

### Temperature & Humidity Sensors
- **TS0201 (ZTH05)** - LCD Temperature & Humidity Sensor

### Motion Sensors
- **ZG-204ZL** - Motion Sensor with Lux
- **ZG-204ZM** - PIR Radar Luminance Sensor (24G)
- **ZG-204ZH** - PIR Radar Luminance Multi-Sensor (24G)
- **ZG-204ZV** - Radar Multi-Sensor (10G)

### Smart Controls
- **Tuya Smart Knob** (TS004F) - Rotary knob with button controls

## Features

- Battery level monitoring
- Temperature and humidity measurements with configurable offsets
- Motion detection with configurable sensitivity
- Radar-based presence detection
- Illuminance measurements
- Smart knob rotation and button press detection
- Multiple operation modes for different use cases

## Installation

1. Install the app from the Homey App Store
2. Add your device through the Homey app
3. Follow the pairing instructions for your specific device

## Compatibility

- Homey Pro (2023) and later
- Requires Homey firmware >= 12.4.0
- SDK 3
