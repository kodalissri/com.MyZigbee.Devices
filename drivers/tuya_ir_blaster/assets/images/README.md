# Device Images

Please replace these placeholder files with actual device images:

- **large.png**: 500x500 pixels, high-quality product image
- **small.png**: 75x75 pixels, thumbnail version

For now, you can use the icon.svg as a temporary placeholder or take photos of the actual Tuya ZS06 IR Blaster device.

To generate placeholder PNGs from the SVG icon, you can use an online converter or ImageMagick:
```bash
convert icon.svg -resize 500x500 large.png
convert icon.svg -resize 75x75 small.png
```
